### Homework 1 : [time-based animation](https://ojrock.github.io/CGI/HW01/hw1.html)
- 技術要求
  - 模型建構 : 選擇合適的配色，XY plane、初始在+Y方向
  - 運動 : 依real-time順時針旋轉，使用於60秒內之計時
  - 不同display style : 扇形
- 功能
  - 三個按鈕可切換 Analog、Sector、Quartz 顯示方式
  - 可以由按鈕暫停(PAUSE, RESTART)
